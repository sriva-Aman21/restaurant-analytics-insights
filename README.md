🍽️ Restaurant Data Analysis Project
📌 Overview

This project focuses on end-to-end data analysis of a restaurant dataset using Python. It covers data cleaning, exploratory data analysis (EDA), visualization, and NLP-based review analysis to extract meaningful insights about cuisines, cities, pricing, ratings, and customer behavior.

🎯 Objectives
Clean and preprocess raw restaurant data
Perform exploratory data analysis (EDA)
Visualize trends and patterns
Analyze customer reviews using NLP
Generate actionable business insights
🛠️ Tech Stack
Python
Pandas, NumPy – Data manipulation
Matplotlib, Seaborn – Data visualization
NLTK / Text Processing – Review analysis
